# Web-Based Report Card System

## Easy-to-Use Report Card Management System

This is a web-based version of your Report Card System that can be used by anyone, even non-IT people.

## How to Use (Super Simple!)

### For Windows Users:
1. **Double-click** `Start_Report_Card_Website.bat`
2. Your browser will open automatically
3. Start using the system!

### For Other Users:
1. Open terminal/command prompt
2. Navigate to this folder
3. Run: `python start_web_app.py`
4. Browser will open automatically

## Features

✅ **Easy Web Interface** - No technical knowledge needed
✅ **Load Excel Data** - Import your existing student data
✅ **Search Students** - Find students quickly by name or roll number
✅ **Filter by Class** - View students by specific classes
✅ **Edit Report Cards** - View and modify all student information
✅ **Teacher Remarks** - Add/edit teacher comments
✅ **Print Reports** - Export and print individual report cards
✅ **Same Format** - Maintains your original report card design
✅ **Works Offline** - No internet required after setup

## How to Import Your Data

1. Click "Load Excel Data" button
2. Select your Excel file (.xlsx or .xls)
3. System will automatically load all students
4. Start managing report cards!

## Excel File Format

Your Excel file should have columns like:
- Student Name
- Class
- Roll Number
- Subject marks (English, Urdu, Mathematics, etc.)
- Computer (Grade: A, B, C, D, F)

## System Requirements

- Any computer with Python installed
- Any web browser (Chrome, Firefox, Edge, Safari)
- Your existing Excel data files
